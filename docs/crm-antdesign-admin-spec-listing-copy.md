# crm-antdesign-admin-spec｜上架文案草案

## 标题
CRM Ant Design Admin Spec

## 副标题
A reusable skill for classifying, structuring, and generating B2B CRM/admin pages with Ant Design-style semantics.

## 简短卖点
Stop turning every admin page into a dashboard. This skill helps AI choose the right page type first, then generate a realistic admin-friendly structure.

## 核心标签
- CRM
- Admin UI
- Ant Design
- Dashboard
- List Page
- Config Page
- AI Analytics
- UI Generation

## 简短介绍
`crm-antdesign-admin-spec` helps AI design and generate B2B CRM/admin pages with stronger structure, better taxonomy, and more realistic Ant Design-style output. It is especially useful for statistics pages, AI analytics pages, list pages, and configuration/tool pages.

## 为什么值得安装
- separates dashboard pages from list/config pages
- improves implementation realism
- adds token/component guidance
- works well with AI page generation workflows
- reduces visual drift in admin systems

## 推荐展示点
1. Page-type taxonomy first
2. Ant Design-oriented layout thinking
3. Token + component mapping
4. Prompt templates for AI generation
5. Private/public adaptation path
