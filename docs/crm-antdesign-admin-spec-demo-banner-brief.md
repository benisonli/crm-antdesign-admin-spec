# crm-antdesign-admin-spec｜Banner / 封面图文案简报

## 目标
为 skill 的发布页准备一张一眼能说明价值的封面图 / banner。

---

## 核心信息
### 主标题（推荐）
**CRM Ant Design Admin Spec**

### 副标题（推荐）
**Classify the page first. Then generate the right admin structure.**

### 中文辅助说明
先判断页面类型，再按 Ant Design 中后台逻辑输出正确页面结构。

---

## 核心卖点（可放在 banner 上）
- Statistics / AI Analytics / List / Config page taxonomy
- Ant Design-oriented admin structure
- Token + component mapping
- Better prompts for AI page generation

---

## 视觉建议
### 风格方向
- B2B admin / CRM / SaaS
- 干净、专业、理性
- 轻科技感，不炫技

### 颜色建议
- 主色：`#1677FF`
- 渐变：`#2586FF -> #2874FC`
- 深色辅助：`#292E33`
- 背景浅色：`#F5F7FA`

### 画面构成建议
Banner 可拆成左右结构：

#### 左侧
- 标题
- 副标题
- 4 个核心卖点标签

#### 右侧
- 4 类页面卡片小样：
  - Statistics
  - AI Analytics
  - List Page
  - Config Page

### 禁止项
- 不要做成赛博朋克
- 不要霓虹发光过重
- 不要营销官网风
- 不要像 C 端 App 宣传页

---

## 可直接给设计师/AI 的一句话
请生成一张用于 AI skill 发布页的 banner，主题为“CRM Ant Design Admin Spec”，风格偏 B2B CRM / SaaS 中后台，主色为蓝色，强调“先判断页面类型，再生成正确后台结构”，右侧展示四类页面卡片缩略形态：Statistics、AI Analytics、List、Config。
